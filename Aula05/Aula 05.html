Criando um Flex Container

O display flex vem sempre como padrão nowrap

O Flex direction define a direção das div flex. Possui as propriedades column, column-reverse, row e row-reverse

O flex-wrap é como o conteudo vai se adaptar a tela. Possui as propriedades wrap, wrap-reverse e nowrap

O Flex-flow é um atalho para definir a Direction e o Wrap

Justify content é utilizado para alinhar os itens flex. Possui os atributos flex-start, flex-end, center, space-around e space-between
